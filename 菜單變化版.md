# 菜單變化版（吃不膩版本）

> 80 天減重計畫專用
> 基礎代謝：2300 kcal
> 每日熱量：1800-2200 kcal（製造 500-800 kcal 赤字）
> 蛋白質目標：180-220g/天
> 乳清蛋白：MARS 水解乳清隨手包（35g/包，26g 蛋白質）

# 調味料搭配建議

## 低熱量調味組合

| 組合名稱 | 材料 | 適合搭配 |
|----------|------|----------|
| **和風** | 醬油 + 味醂 + 薑 | 雞肉、魚 |
| **檸檬香草** | 橄欖油 + 檸檬 + 迷迭香 | 雞肉、海鮮 |
| **蒜香** | 蒜末 + 橄欖油 + 鹽 | 任何肉類 |
| **泰式** | 魚露 + 檸檬 + 辣椒 | 雞肉、蝦 |
| **墨西哥** | 孜然 + 辣椒粉 + 蒜 | 牛肉、雞肉 |
| **韓式** | 辣椒醬 + 蒜 + 芝麻油 | 豬肉、雞肉 |
| **地中海** | 橄欖油 + 奧勒岡 + 番茄 | 魚、雞肉 |
| **日式味噌** | 味噌 + 味醂 + 清酒 | 鮭魚、茄子 |
